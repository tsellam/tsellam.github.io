Secret Sauce datasets - Guitar Effects

Description:

The aim of Secret Sauce is to automatically reverse engineer guitar and synth sounds, and thus answer the question "Wow, how did they make that sound?!".

This data set contains 10,000 sound samples made of 1 note (C3) played with a simulated guitar routed to an amp simulator, a distortion, a delay, a chorus/flanger and a reverb. We report the setting that we used to create the sounds in the file settings.csv. We normalized the continuous variables to 0-10. The aim is to predict the settings based on the sounds.

The guitar sound was created with Ableton tension and the effects were produced with NI Guitar Rig 5.

The samples last 840ms, they are mono encoded in 16 bits/22,050Hz. We did not normalize the volume, but we did use a limiter to avoid saturation.

More info (here) [sellam.me/2018/01/01/Secret-Sauce-data.html]

Files:

  • settings.csv: the target variables to predict.
  • samples: sample sounds
  • GuitarRigPlugin.ngrr: plugin preset, necessary if you wish to reproduce the sounds

What do the column names means?

Each column corresponds to a parameter of the original plugin/synthesizer. You may find help here.
